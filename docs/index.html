SCHOLARは、新しい価値やサービスを作る人（研究開発職、デベロッパー、プログラマー、新規事業開発担当者など）
のための極めてレベルの高い講義とディスカッションを提供します。
講師は現場で活躍する超一流のサイエンティストもしくはエンジニアです。

「どんな内容の研究をしているか」だけではなく「それをどういう方法で研究しているか」の手法（メソッド）の解説を中心とすることで、講義を受けた人がすぐに実践できる方法論、触発される考え方を提供します。

講義はテーマが異なる8つのセクションにより構成、テーマについての解説＋演習（グループディスカッション）を8回繰り返します。

講師はPowerpointやKeyNoteなどでつくられた資料を一切利用せず、HandWriting（手書き）によるスクーリングによってナラティブな授業を提供します。

参加者はネットで講義を繰り返し参照可能です。またネットコミュニティでのディスカッションや、ネットコミュニティだけの参加も可能です。
